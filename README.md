# pinterest-client
small node app for pinterest

## The tests folder
Automated tests and files to run on CI

### postman folder
postman folder contains the following structure
```
postman
  |
  |-- collections
      | 
      |-- collection_name.json
  |
  |-- environments
      |
      |-- environment_name.json
  |
  |-- data-files
      |
      |-- data-file_name.json
      |-- data-file_name.csv
  |
  |-- reports
      |
      |- report_name.json
      |- report_name.html
  
```
